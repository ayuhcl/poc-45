Welcome to AWS Amplify POC

This static website is hosted using AWS Amplify with automated CI/CD integration from GitHub.

Project Overview

The objective of this project is to host a static website, enable branch-based deployments, monitor traffic using CloudWatch, send alerts through SNS, and restrict access to India using AWS WAF.

Services Used

  • AWS Amplify
  • GitHub
  • Amazon CloudWatch
  • Amazon SNS
  • AWS WAF

Features

  • Static Website Hosting
  • GitHub-based CI/CD
  • Dev, Staging, and Production Environments
  • CloudWatch Monitoring
  • SNS Notifications
  • India-Only Access Restriction
